# wiki-init

Initialize the wiki vault structure.

## What it does

1. Check if wiki/ directory exists
2. If not, create the full wiki structure
3. Create index.md, hot.md, log.md, overview.md
4. Create subdirectories for sources, entities, concepts, domains, questions
5. Report the created structure

## Usage

```
/wiki-init
```
