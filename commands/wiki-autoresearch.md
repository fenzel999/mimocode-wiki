# wiki-autoresearch

Perform autonomous research on a topic and file results to wiki.

## What it does

1. Decompose topic into search angles
2. Execute web searches
3. Fetch and analyze results
4. Create wiki pages for sources, entities, concepts
5. Cross-reference all pages
6. Update index and log

## Usage

```
/wiki-autoresearch .NET 10 new features
/wiki-autoresearch Blazor best practices
/wiki-autoresearch microservices architecture
```
