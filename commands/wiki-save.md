# wiki-save

Save the current conversation as a wiki page.

## What it does

1. Analyze the conversation
2. Determine note type (synthesis, concept, source, question)
3. Create wiki page with frontmatter
4. Update index.md
5. Update log.md
6. Update hot.md

## Usage

```
/wiki-save
/wiki-save "Note Title"
```
