# canvas

Manage the visual knowledge canvas.

## What it does

1. Create or open canvas
2. Add notes, text, images to canvas
3. Manage zones
4. Display canvas status

## Usage

```
/canvas
/canvas new "Research"
/canvas add note [.NET 10]
/canvas add text "Key insight"
/canvas zone "Product"
/canvas list
```
