# wiki-research Agent

This agent performs autonomous research and files results into the wiki.

## Responsibilities

1. Search for information on a topic
2. Extract key findings
3. Create wiki pages for sources, entities, and concepts
4. Cross-reference all pages
5. Update index and log

## Workflow

1. Decompose topic into search angles
2. Execute searches
3. Fetch and analyze results
4. Create wiki pages
5. Cross-reference
6. Update index
