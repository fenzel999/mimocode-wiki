# .NET 10 Overview

## Summary

.NET 10 是 .NET 9 的后续版本，作为长期支持 (LTS) 版本，支持周期为 3 年。

## Key Features

### Runtime
- JIT 内联改进
- 方法去虚拟化
- 栈分配优化
- AVX10.2 支持
- NativeAOT 增强
- 结构体参数代码生成改进
- 循环反转优化

### Libraries
- 密码学新 API
- 全球化支持
- 数值计算
- 序列化改进
- 集合操作
- 诊断工具
- ZIP 文件处理
- 后量子密码学支持
- WebSocketStream 支持
- TLS 1.3 macOS 支持

### SDK
- Microsoft.Testing.Platform 支持
- CLI 命令标准化
- 容器镜像创建
- 平台特定 .NET 工具
- 一次性工具执行
- CLI 内省

### Aspire
- Aspire 13.1 更新

### ASP.NET Core
- Blazor 改进
- OpenAPI 增强
- Minimal API 更新

### C# 14
- 扩展成员
- 空条件赋值
- nameof 支持未绑定泛型类型
- Span<T> 隐式转换
- Lambda 参数修饰符
- field 支持的属性
- partial 事件和构造函数

## Notes

.NET 10 是一个重要的 LTS 版本，包含大量性能改进和新 API。对于 .NET 架构师来说，重点关注：
1. NativeAOT 改进对部署的影响
2. 后量子密码学对安全架构的影响
3. Aspire 13.1 对云原生开发的支持
