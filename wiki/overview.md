# Wiki Overview

**This Wiki is AI's persistent memory.**

## Purpose

1. Provide persistent context for AI across sessions
2. Record project decisions
3. Accumulate knowledge
4. Support visual knowledge graphs

## How It Works

```
Session start → Read hot.md → Restore context → Work → Update Wiki
    ↑                                                    ↓
    └────────────────────────────────────────────────────┘
```

## Key Files

| File | Purpose |
|------|---------|
| `hot.md` | Recent context (read first) |
| `index.md` | Master catalog |
| `overview.md` | This file |
| `log.md` | Operation history |
