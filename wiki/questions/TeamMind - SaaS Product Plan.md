# TeamMind - AI 团队知识持久化平台

## 产品定义

**TeamMind** 是一个 AI 驱动的团队知识管理平台，帮助技术团队持久化和共享知识，消除知识孤岛。

## 目标用户

- 中小型技术团队（5-50 人）
- 远程/分布式团队
- 有新人入职需求的团队
- 技术债务严重的团队

## 核心价值

1. **知识持久化** - 人员流动不丢失知识
2. **AI 自动维护** - 减少人工维护成本
3. **快速上手** - 新人自助学习
4. **一致性** - 团队保持统一标准

## 技术架构

```
┌─────────────────────────────────────────┐
│              Blazor Server              │
│            (.NET 10 + Aspire)           │
├─────────────────────────────────────────┤
│              ASP.NET Core               │
│            Minimal APIs                 │
├──────────┬──────────┬──────────────────┤
│ Postgres │  Redis   │  Meilisearch     │
├──────────┴──────────┴──────────────────┤
│         Semantic Kernel (AI)            │
│     OpenAI / 开源模型 (Llama)          │
└─────────────────────────────────────────┘
```

## 功能模块

### Phase 1: 核心知识库（4 周）

- 知识条目 CRUD
- 分类和标签系统
- AI 智能问答
- 全文搜索
- 版本历史

### Phase 2: 代码集成（3 周）

- GitHub/GitLab 集成
- 代码库知识提取
- AI 代码审查建议
- 规范自动检查

### Phase 3: 协作功能（2 周）

- 团队权限管理
- 知识审批流程
- 评论和讨论
- 通知系统

### Phase 4: 部署上线（1 周）

- Docker 容器化
- Aspire 编排
- CI/CD 流水线
- 监控和日志

## 成本预估

| 项目 | 月成本 |
|------|--------|
| 服务器 (Azure/AWS) | $50-100 |
| AI API | $20-50 |
| 数据库 | $0-30 |
| 域名/SSL | $10 |
| **总计** | **$80-190/月** |

## 变现策略

### 阶段 1: 开源（0-6 个月）
- GitHub 开源
- 建立社区
- 收集反馈

### 阶段 2: 托管服务（6-12 个月）
- 提供托管版本
- 免费层 + 付费层
- $10-50/用户/月

### 阶段 3: 企业版（12+ 个月）
- 企业级功能
- 私有部署
- 定制开发

## 风险评估

| 风险 | 影响 | 应对策略 |
|------|------|---------|
| 用户获取难 | 高 | 开源 + 社区运营 |
| 维护成本高 | 中 | AI 自动化 + 社区贡献 |
| 竞争激烈 | 中 | 聚焦细分市场 |
